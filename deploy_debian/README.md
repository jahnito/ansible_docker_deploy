Role Name
=========

DEPLOY_DEBIAN

This role is intended to be distributed on Debian Linux

Requirements
------------

OS requirements

To install Docker Engine, you need the 64-bit version of one of these Debian versions:

    Debian Bookworm 12 (stable)
    Debian Bullseye 11 (oldstable)
